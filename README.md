# Someday team's Web Front-End Projects Template Repository.

### 커밋 메세지
* feat: 새로운 기능에 대한 커밋
* fix: 버그 수정에 대한 커밋
* build: 빌드 관련 파일 수정에 대한 커밋
* chore: 그 외 자잘한 수정에 대한 커밋
* ci: CI관련 설정 수정에 대한 커밋
* docs: 문서 수정에 대한 커밋
* style: 코드 스타일 혹은 코드 포맷에 관한 커밋
* refactor: 코드 리팩토링에 대한 커밋
* test: 테스트 코드 수정에 대한 커밋
